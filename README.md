## Help
Use Python 3.6.8

In windows, i use python portable and i create a Jarvis.bat with:

	@echo off
	C:\Jarvis\Python_Win\App\python.exe C:\Jarvis\Jarvis.py %*

and i save in C:\Windows\System32.

So...

C:\Users\Usuario>jarvis help
